## 연결 상태

<details markdown="1" open>
<summary><strong>연결 상태 값</strong></summary>

| 상태 | 설명 |
|------|------|
| `connect` | ZDM 센터에 정상적으로 연결됨 |
| `disconnect` | ZDM 센터 연결 끊김 |

</details>

## 활성화 상태

<details markdown="1" open>
<summary><strong>활성화 상태 값</strong></summary>

| 상태 | 설명 |
|------|------|
| `ok` | ZDM 센터가 정상적으로 활성화됨 |
| `fail` | ZDM 센터 활성화 실패 |

</details>
